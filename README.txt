# Ice Cream Run

This project is a game developed using Java and the LibGDX framework. The game involves a truck that moves around a grid, with the ability to interact with certain nodes on the grid.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have the following installed:

- Java
- Gradle
- Python
- JavaScript
- C
- C++
- pip

### Installing

Clone the project from the GitHub repository:

```
git clone https://github.com/cs-440-at-uic/442-Group-8-Spring-2024.git
```

Navigate to the project directory and build the project using Gradle:

```
cd 442-Group-8-Spring-2024
gradle build
```

## Built With

- [Java](https://www.java.com/) - The main programming language used.
- [LibGDX](https://libgdx.badlogicgames.com/) - The game development framework used.
